# Gsit
一个加入各种丰富动作的插件
[源地址](https://www.spigotmc.org/resources/gsit-modern-sit-seat-and-chair-lay-and-crawl-plugin-1-13-1-20.62325/)

# 使用文档
```minecraft command
/gbellyflop | /bellyflop: 翻肚皮?
/gcrawl | /crawl: 趴下
/gemote | /emote: 没什么卵用的表情
/glay | /lay: 躺平了
/gsit | /sit: 坐下！
/gspin | /spin: 龙卷风摧毁停车场！
```
