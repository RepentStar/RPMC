# Geyser

Geyser 让 MCBE(Minecraft: Bedrock Edition)玩家能在 MCJE(Minecraft: Java Edition)的服务器上欺负腐竹。其允许 BE 玩家使用 JE 服务器上的一些功能，例如命令、皮肤和特性。

[插件发布地址](https://geysermc.org/)

# 使用文档

本服已经启用 Geyser

连接时需要填写的 IP: 在本服 JAVA 版 IP 前加上`ip.`即可

端口不用改，用默认的

-   认证模式:离线模式
-   代码补选:√
-   允许第三方皮肤/耳朵(?): √
-   攻击冷却条显示: actionbar 位置
-   显示坐标: √
-   快捷换盔甲: √
