# ExcellentEnchants
更好的附魔，添加许多额外正/负向附魔

所有可以在原版附魔上使用的操作，在这里都可以使用

会自然生成附魔书，可以与村民交易得到，可以钓鱼钓到……

[源地址](https://www.spigotmc.org/resources/excellentenchants-vanilla-like-enchantments.61693/)
